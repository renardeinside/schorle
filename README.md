\*\*`Schorle` (pronounced as [ˈʃɔʁlə](https://en.wikipedia.org/wiki/Schorle)) is the Foundation Layer for your Data and AI Applications.

It combines Pythonic backend with simple and easy to work with JSX/CSS setup.
