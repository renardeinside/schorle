## New Project

```
uvx run schorle init my-app
```

## Existing Project

Schorle can be easily added to existing projects:

```bash
uv add schorle
uvx run schorle init <package_name>.ui --app-only
```

## Add the UI

<CodeEditor

As a result, the `index` page will be rendered on each request to the `/` endpoint.

  // server-side parameter passing
  const { title, userCount } = useParams();

return (
<div>
<h1>{title}</h1>
<p>Join {userCount} developers</p>
</div>
);
}`}
</CodeEditor>

## Start Development Server

```bash
schorle dev
```

This will:

- Start the development server on `http://localhost:8000`
- Automatically compile your TypeScript/React components
- Generate Python bindings for your UI components
- Enable hot reloading for both Python and TypeScript code

## Project Structure

After initialization, your project will have this structure:

```
my-app/
├── src/
│   └── my_app/
│       ├── app.py          # FastAPI backend
│       └── ui/             # Frontend code
│           ├── components/ # React components
│           ├── pages/      # Page components
│           └── lib/        # Utilities & API hooks
├── pyproject.toml         # Python dependencies
└── README.md
```
